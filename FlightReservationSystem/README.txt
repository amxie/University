A simple program designed to help with reserving and canceling randomly generated flights.
The program supports regular flights along with Long Haul Flights, along with sorting by departure time, duration of the flight, and sorting the types of aircrafts.
This also inclues the features of checking if seats for a flight is available, and reserving first class seats only for long haul flights.

Covers the function of res and cancel for a specific seat number, will then display in seats if the seat is occupied or not
Class Flight throws exceptions for any problems
Flights have been changed to a tree map
PASMAN function was added to display a list of passengers in a flight
